# HowToEdibles — Static Site

Cannabis edibles recipe site hosted on GitHub Pages at www.howtoedibles.com.

## Architecture

Fully static HTML site — no build framework, no bundler. Pages are either hand-maintained (English root articles/homepage) or generated by `scripts/build_site.rb`.

- **Hosting:** GitHub Pages (branch: master)
- **Domain:** www.howtoedibles.com (CNAME file)
- **Languages:** en (root), pt, es, de, zh — generated pages live in `/{lang}/`
- **CSS:** Bootstrap 4.5.2 (CDN) + `/css/styles.css` (3.3k lines, custom)
- **JS:** Vanilla JS in `/js/` — no build step, no modules
- **Images:** `/images/recipes/{slug}.jpg` — AI-generated via Gemini or SVG placeholders

## Key Commands

```bash
# Rebuild all generated pages (translated homepages, recipes, articles, sitemap)
ruby scripts/build_site.rb

# Generate placeholder images for recipes without photos
python3 scripts/generate_placeholders.py

# Generate real recipe photos via Gemini API (needs GEMINI_API_KEY env var)
python3 scripts/generate_images_gemini.py

# Translate recipe text (ingredients/instructions) via term mapping
ruby scripts/translate_recipes.rb
```

## Conventions

- English root HTML files are **hand-maintained** — `build_site.rb` never overwrites them
- Recipe pages for ALL languages (including English) are generated by `build_site.rb`
- Dosage formula: `dosage_mg = (10 * grams * strength_percent) / num_portions`
- Recipe URLs: `/recipes/{slug}/` (directory with index.html)
- Calculator reads `window.RECIPE_DEFAULTS` (injected inline per recipe page)
- Dark mode uses `[data-theme="dark"]` CSS selectors, persisted in localStorage
- Categories: Desserts, Drinks, Snacks, Essentials, Lunch, Keto, Vegan, International

## Data Flow

1. `data/recipes.json` is the single source of truth for recipe metadata
2. `data/recipes_text.json` has full ingredients/instructions
3. `data/recipe_enrichment.json` has tips, variations, pairings, FAQ
4. `i18n/*.json` has all translations
5. `scripts/build_site.rb` combines all of the above → generates HTML
